# Chào mừng đến Kho Prompt Docs

Kho Prompt là marketplace dành cho prompt AI, prompt kit và automation flow được vận hành cho cộng đồng Việt Nam. Bộ tài liệu này cung cấp hướng dẫn ngắn gọn, dễ đọc dành cho người dùng cuối và tác giả nội dung.

## Bạn sẽ tìm thấy gì?

- Tổng quan sản phẩm: Prompt, Prompt Kit và Automation Flow hoạt động ra sao.
- Hướng dẫn bán prompt: quy trình gửi nội dung, theo dõi duyệt và nhận tiền.
- Chính sách phí nền tảng: cách tính seller amount, platform fee và ví tác giả.

## Đối tượng phù hợp

- Người mới đang tìm cách khai thác kho nội dung sẵn có.
- Tác giả muốn bán prompt hoặc workflow và cần hiểu chính sách chi trả.
- Đội hỗ trợ khách hàng cần câu trả lời nhanh cho người dùng.

Bấm vào từng mục trong thanh điều hướng hoặc `SUMMARY.md` để bắt đầu.
